Fuentes Google

Como añadir Fuentes

Instalar una fuente de Google Fonts es fácil. Utilizaremos como ejemplo la fuente Roboto y Outfit. Lo haremos paso a paso:

Primer paso, seleccionar la fuente.

Segundo paso, seleccionar la fuente.

Tercer paso, instalación standard.

Cuarto paso, instalación @import.

INSTALACIÓN STANDARD

Añadiremos a nuestro html entre la etiqueta el siguiente link

<link href="https://fonts.googleapis.com/css?family=Modak|Roboto" rel="stylesheet">

INSTALACIÓN @IMPORT

Esta es la forma más recomendada de hacerlo. Insertamos en primera línea donde tengamos nuestro CSS el código generado

@import url('https://fonts.googleapis.com/css?family=Outfit|Roboto');

UTILIZACIÓN

body{ font-family: 'Roboto', sans-serif; }
