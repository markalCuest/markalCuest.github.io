ICONOS GOOGLE

Como añadir iconos

Para poder insertar los iconos de Google en nuestra pagina web, necesitaremos el siguiente link a la hoja de estilos

<link rel="stylesheet" href="https://fonts.googleapis.com/icon?family=Material+Icons">

Para poder insertar un icono en nuestra pagina web tendremos que utilizar la etiqueta <i> con el atributo class al cual asignaremos "material-icons" para referirnos al icono

Si queremos utilizar un icono en concreto tendremos que ponerle dentro de la etiqueta el nombre deseado

Algunos ejemplos de iconos:

fingerprint bug_report accessibility_new cancel

Editar Iconos

Para poder editar un icono tendremos que agregar el atributo Style y lo que se quiera añadir

Ejemplos:


Tamaño de letra:

traffic

<i class="material-icons" style="font-size: 24px;"> traffic </i>


Tamaño de letra y color:

cloud

<i class="material-icons" style="font-size:35px;color:lightblue;"> cloud </i>


Tamaño de letra, color y sombras

favorite

<i class="material-icons" style="font-size:30px;color:lightblue;text-shadow: 2px 2px 5px #000000;"> favorite </i>


COMBINACIONES DE ESTILOS

Tambien se pueden realizar distintas combinaciones, por ejemplo podemos combinar un icono con un elemento button

Para poder realizar una combinacion tendremos que definir un elemento cualquiera e introducir distintos atributos de estilo

Ejemplo:

Para poder editar un icono a un boton primero crearemos el botó mediante las etiquetas <Button>

Si queremos decorarlo tendremos que agregar el atributo Style y lo que se quiera añadir


Tamaño de letra y color:


<button class="glyphicon glyphicon-cloud style="font-size:18px;color:red;"></button>
