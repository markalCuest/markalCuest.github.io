Iconos Google

Como añadir iconos

Para poder insertar los iconos de Google en nuestra pagina web, necesitaremos el siguiente link a la hoja de estilos

<link rel="stylesheet" href="https://fonts.googleapis.com/icon?family=Material+Icons">

Para poder insertar un icono en nuestra pagina web tendremos que utilizar la etiqueta <i> con el atributo class al cual asignaremos "material-icons" para referirnos al icono

Si queremos utilizar un icono en concreto tendremos que ponerle dentro de la etiqueta el nombre deseado

Algunos ejemplos de iconos:

fingerprint bug_report accessibility_new cancel

Editar Iconos

A la hora de editar un icono tendremos que agregar el atributo Style y dentro los parametros que quieras darle al icono

- Un tamaño mas grande o mas pequeño

fingerprint

utilizaremos el atributo font-size <i class="material-icons" style="font-size"> bug_report </i>

- Cambiar el color:

bug_report

<i class="material-icons" style="font-size:35px;color:lightblue;"> bug_report </i>

- Podemos añadirle tambien sombras a nuestro icono

bug_report

<i class="material-icons" style="font-size:30px;color:lightblue;text-shadow: 2px 2px 5px #000000;"> bug_report </i>

No solo podemos modificar el estilo de los iconos, si no que tambien podemos convertirlo en un botón, enlace..
