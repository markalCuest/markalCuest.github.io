Iconos Font Awesome

Como añadir iconos

Para poder insertar los iconos de Font Awesome en nuestra pagina web, necesitaremos el siguiente link a la hoja de estilos

<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/4.7.0/css/font-awesome.min.css">

Para poder insertar un icono en nuestra pagina web tendremos que utilizar la etiqueta <i> con el atributo class al cual asignaremos "fa fa-cloud" para referirnos al icono

Si queremos utilizar un icono en concreto tendremos que ponerle dentro de le etiqueta class despues de la primera clase el icono que queremos utilizar, en este caso anterior seria fa-cloud

Editar Iconos

A la hora de editar un icono tendremos que agregar el atributo Style y dentro los parametros que quieras darle al icono

- Un tamaño mas grande o mas pequeño


utilizaremos el atributo font-size <i class="fa fa-headphones" style="font-size"> </i>

- Cambiar el color:


<i class="fa fa-address-card" style="font-size:35px;color:lightblue;"></i>

- Podemos añadirle tambien sombras a nuestro icono

<i class="fa fa-telegram" style="font-size:30px;color:lightblue;text-shadow: 2px 2px 5px #000000;"></i>

No solo podemos modificar el estilo de los iconos, si no que tambien podemos convertirlo en un botón, enlace..
