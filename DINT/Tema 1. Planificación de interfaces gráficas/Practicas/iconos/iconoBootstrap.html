Iconos Bootstrap

Como añadir iconos

Para poder insertar los iconos de Bootstrap en nuestra pagina web, necesitaremos el siguiente link a la hoja de estilos

<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/css/bootstrap.min.css">

Para poder insertar un icono en nuestra pagina web tendremos que utilizar la etiqueta <i> con el atributo class al cual asignaremos "glyphicon glyphicon-eye-open" para referirnos al icono

Si queremos utilizar un icono en concreto tendremos que ponerle dentro de le etiqueta class despues de la primera clase el icono que queremos utilizar, en este caso anterior seria glyphicon-eye-open

Editar Iconos

A la hora de editar un icono tendremos que agregar el atributo Style y dentro los parametros que quieras darle al icono

- Un tamaño mas grande o mas pequeño


utilizaremos el atributo font-size <i class="glyphicon glyphicon-bitcoin" style="font-size"> </i>

- Cambiar el color:


<i class="glyphicon glyphicon-cd" style="font-size:35px;color:lightblue;"></i>

- Podemos añadirle tambien sombras a nuestro icono

<i class="glyphicon glyphicon-hand-right" style="font-size:30px;color:lightblue;text-shadow: 2px 2px 5px #000000;"></i>
